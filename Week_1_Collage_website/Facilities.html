Central Library

The Central Library is one of the important facilities of Prasad V Potluri Siddhartha Institute of Technology as it caters to the information and research needs of faculty and students. Originated in the year 1998. The Central Library has adopted Open Access System. All the books are arranged classified as per Dewey decimal classification 21st edition which is helpful to the users to locate his / her desired book. Central Library is Fully Automated using LIBSYS10 LMS (Library Management Software). It is Web Centric having support for Twenty Thousand Unique Records with five Concurrent housekeeping clients along with twenty five concurrent users for web OPAC (On â€“ line Public Accessing Catalogue) on server Under Windows 2012 platform. This Software is supporting Barcode system for issue & return of books. The Library is an Institutional member of DELNET. It provides web access to 1,21,08,793 records of Books, Periodicals, Thesis, Dissertations. It is connected with 2196 libraries.
